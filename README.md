# motivational-clock-app
React-додаток, що показує поточний час та випадкові мотиваційні цитати з анімованим фоном.

# Додаток "Мотиваційний годинник"

Простий веб-додаток на React, який відображає поточний час та випадкові мотиваційні цитати.  
Додаток адаптивний для мобільних пристроїв і має плавний анімований градієнтний фон для сучасного вигляду.

# Особливості
- Живий годинник, що оновлюється кожну секунду.
- Отримання випадкових мотиваційних цитат через [API Ninjas](https://api.api-ninjas.com/api/advice).
- Кнопка 'Get New Quote' для завантаження нової цитати.
- Фіксована висота контейнера цитат для стабільного інтерфейсу.
- Адаптивний дизайн для мобільних та десктопних пристроїв.
- Легкі hover-ефекти та анімований градієнтний фон для кращого UX.

# Технології
- React.js (класові компоненти)
- CSS3 (Flexbox, анімації)
- Fetch API
